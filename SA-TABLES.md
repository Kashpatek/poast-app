# SA Brand · Excel reference thumbnails

The chart-maker-2 quick-start gallery renders a small "📊 Reference original" link on every SA Brand template tile. Clicking it opens a lightbox showing the original SemiAnalysis brand-template Excel sheet rasterized to PNG. The PNGs are produced locally by [`scripts/rasterize-sa-tables.mjs`](scripts/rasterize-sa-tables.mjs), which shells out to LibreOffice headless — we deliberately do NOT commit the PNGs, since they're derivatives of the brand assets and easy to regenerate. To populate the thumbnails, install LibreOffice once (`brew install --cask libreoffice` on macOS) and then run `npm run rasterize-tables` from the repo root. The script walks `~/Desktop/SEMIANALYSIS/Brand/` (override via the `SA_TABLE_REFS_DIR` env var), drops the resulting PNGs in `public/sa-table-refs/`, and exits 0 cleanly even when LibreOffice is missing — so the Next.js build never breaks on its absence.
