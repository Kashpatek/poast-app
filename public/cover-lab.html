How this works

  • Each cover is a 1080×1350 SVG that re-renders whenever you change a control above.
  • Click FULLSCREEN on any card to inspect at native size. SVG downloads the file ready for InDesign / Illustrator. PNG rasterizes via canvas.
  • Tell me which numbers you want as cover-picker options in the verbatim carousel flow and I'll port them into carousel.tsx. The SVGs here become the React renderers — same paths, same math.
  • Variants that don't use the cover image (02 · Type Bomb) still render so you can compare type weight against the photo treatments.
  • 03 · Diptych defaults to one full image. Tick DUAL IMAGE in the image controls to split it horizontally — top half stays color, bottom half flips to an accent-tinted duotone with a hairline accent seam between.
